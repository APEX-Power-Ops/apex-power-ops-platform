# Knowledge Lane

This directory holds the active governed knowledge surfaces that have been physically landed into the unified platform root.

Current imported slices are intentionally bounded:

1. authority and standards surfaces under `authority/`
2. build and publication specs under `specs/`
3. reference workflow guides under `process-guides/`

The first landing is provenance-preserving rather than fully rewritten.

Use the imported documents as the active knowledge-lane baseline inside the unified root, but treat source-repo-relative startup instructions or broader workspace references as historical context unless later normalization packets explicitly replace them.

Heavy source-library holdings, archive-heavy material, full `Development/` trees, and published content trees remain outside this first landing by design.
