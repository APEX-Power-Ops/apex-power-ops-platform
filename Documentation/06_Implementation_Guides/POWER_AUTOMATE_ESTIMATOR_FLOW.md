# Power Automate Flow: Estimator JSON Import

**Updated:** November 29, 2025  
**Status:** Active Specification

---

## Overview

This flow imports Estimator data from JSON files into Dataverse. JSON files are generated by the VBA macro `DataverseExport.bas` in the Estimator Excel workbook.

**See also:** [ESTIMATOR_FLOW_SPECIFICATION.md](./ESTIMATOR_FLOW_SPECIFICATION.md) for complete architecture details.

---

## Flow Definition

### Trigger
**When a file is created (properties only)**
- Site Address: SharePoint site with synced folder
- Library Name: `Documents`
- Folder: `/Dataverse_Imports/`
- Filter: `*.json` files only

### Flow Diagram

```
┌─────────────────────────────────────────────────────────────────┐
│ 1. TRIGGER: JSON file created in /Dataverse_Imports/           │
│    Condition: File ends with .json                              │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 2. GET FILE CONTENT                                             │
│    Action: Get file content                                     │
│    Site: SharePoint site                                        │
│    File: @{triggerOutputs()?['body/Id']}                       │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 3. PARSE JSON                                                   │
│    Content: @{body('Get_file_content')}                        │
│    Schema: See JSON Schema section below                        │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 4. LOOKUP/CREATE CLIENT                                         │
│    Query Dataverse: cr950_clients                               │
│    Filter: cr950_name eq '@{body('Parse_JSON')?['client']?['name']}'
│    If not found → Create new client                             │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 5. LOOKUP/CREATE SITE                                           │
│    Query Dataverse: cr950_locations                             │
│    Filter: cr950_name eq '@{body('Parse_JSON')?['site']?['name']}'
│    If not found → Create with address, city, state, zip         │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 6. CREATE/UPDATE PROJECT                                        │
│    Query: Existing by projectNumber?                            │
│    Create/Update cr950_projectses with lookups                  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 7. FOR EACH SCOPE                                               │
│    Apply to each: @{body('Parse_JSON')?['scopes']}             │
│    ├─ Create Scope in cr950_scopeses                           │
│    └─ FOR EACH APPARATUS                                        │
│       └─ Create Apparatus in cr950_apparatus                    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│ 8. MOVE PROCESSED FILE                                          │
│    Move to /Dataverse_Imports/Processed/                        │
└─────────────────────────────────────────────────────────────────┘
```

## Dataverse Actions

### Create Client (if not exists)
```
Table: cr950_clients
Fields:
  - cr950_name: @{body('Parse_JSON')?['client']?['name']}
```

### Create Site (if not exists)
```
Table: cr950_locations
Fields:
  - cr950_name: @{body('Parse_JSON')?['site']?['name']}
  - cr950_address: @{body('Parse_JSON')?['site']?['address']}
  - cr950_city: @{body('Parse_JSON')?['site']?['city']}
  - cr950_state: @{body('Parse_JSON')?['site']?['state']}
  - cr950_zipcode: @{body('Parse_JSON')?['site']?['zipCode']}
  - cr950_contactname: @{body('Parse_JSON')?['site']?['contactName']}
  - cr950_contactemail: @{body('Parse_JSON')?['site']?['contactEmail']}
```

### Create/Update Project
```
Table: cr950_projectses
Fields:
  - cr950_name: @{body('Parse_JSON')?['project']?['name']}
  - cr950_jobnumber: @{body('Parse_JSON')?['project']?['projectNumber']}
  - cr950_projectlead: @{body('Parse_JSON')?['project']?['projectLead']}
  - cr950_startdate: @{body('Parse_JSON')?['project']?['startDate']}
  - cr950_quotedamount: @{body('Parse_JSON')?['summary']?['grandTotal']}
  - cr950_clientid: [Client lookup ID]
  - cr950_locationid: [Site lookup ID]
```

### Create Scope (for each)
```
Table: cr950_scopeses
Fields:
  - cr950_name: @{items('For_each_scope')?['name']}
  - cr950_scopetype: @{items('For_each_scope')?['scopeType']}
  - cr950_totalhours: @{items('For_each_scope')?['totalHours']}
  - cr950_quotedamount: @{items('For_each_scope')?['quotedAmount']}
  - cr950_onsitelabor: @{items('For_each_scope')?['financials']?['onsiteLaborTotal']}
  - cr950_offsitelabor: @{items('For_each_scope')?['financials']?['offsiteLaborTotal']}
  - cr950_travel: @{items('For_each_scope')?['financials']?['travelTotal']}
  - cr950_outsideservices: @{items('For_each_scope')?['financials']?['outsideServicesTotal']}
  - cr950_projectid: [Project lookup ID]
```

### Create Apparatus (for each in scope)
```
Table: cr950_apparatus
Fields:
  - cr950_apparatustype: @{items('For_each_apparatus')?['equipmentType']}
  - cr950_quantity: @{items('For_each_apparatus')?['quantity']}
  - cr950_hoursperunit: @{items('For_each_apparatus')?['hoursPerUnit']}
  - cr950_totalhours: @{items('For_each_apparatus')?['totalHours']}
  - cr950_section: @{items('For_each_apparatus')?['section']}
  - cr950_scopeid: [Scope lookup ID]
  - cr950_projectid: [Project lookup ID]
```

## Error Handling

| Error | Action |
|-------|--------|
| JSON parse failure | Log error, notify admin, move to /Failed/ |
| Client lookup fails | Create new client record |
| Site lookup fails | Create new site record |
| Project already exists | Update existing record |
| Dataverse timeout | Retry 3x with exponential backoff |
