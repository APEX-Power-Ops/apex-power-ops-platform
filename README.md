## 🎯 Overview


Full-stack operations platform for electrical field testing projects. Tracks apparatus-level work (switchgear, transformers, breakers) through the complete project lifecycle with automated revenue recognition, NETA-aligned compliance workflows, and Power System Studies (PSS) coordination.
