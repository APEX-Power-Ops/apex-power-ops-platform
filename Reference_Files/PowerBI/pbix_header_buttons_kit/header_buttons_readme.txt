This kit contains header images for your Power BI dashboard. To use them:

1. Choose a header image and save it locally.  The images have a 16:9 aspect ratio (1600×80 px) and use your brand colours.

2. In Power BI Desktop, open your report or start a new one.  On each page, go to **Insert → Image** and select one of the header images.  Resize it to span the top of the canvas.

3. Create navigation buttons by going to **Insert → Buttons → Blank**.  Place them on the header and label them (e.g., CHART, MATRIX, GANTT, REPORT).  In the **Format** pane, set the Fill colour, border radius, and font to match your style.  Use the **Action** property to link each button to a report page or bookmark.

4. If you want filters like in the sample mock‑up (All Scopes, All Status, etc.), add **Slicer** visuals below the header.  Use slicer settings to change them to drop‑down or tile style.

5. Add your card, chart or matrix visuals below the slicers.  Use your DAX measures for Total Tasks, Completed Tasks, etc.  Arrange them in rows to match the mock‑up.

6. When you’re satisfied with the layout, save your report as a `.pbix` file or export it as a template (`.pbit`).

The images in this kit are:

* `header_logo_text_1600x80.png` – Dark gradient with the full RESA Power Service logo (logo plus text).
* `header_gradient_logo_final_1600x80.png` – Dark gradient with only the logo symbol, no text.
* `gradient_header_1600x80.png` – Plain dark gradient with no logo or text.
